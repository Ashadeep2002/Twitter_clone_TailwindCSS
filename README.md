# Twitter Clone (TailwindCSS)

A simple and responsive **Twitter UI clone** built using **HTML + Tailwind CSS**.  
Designed to practice modern layouts, utility-first styling, and clean UI structure.

---

## 🚀 Features
- Fully responsive Twitter-style layout  
- Clean feed, sidebar & trend sections  
- Pure TailwindCSS (no custom CSS needed)  
- Easy to extend into React/Next.js  

---

## 🔥 Live Demo- https://tweetfeed.netlify.app/

---

## 🛠️ Tech Stack
- **HTML**
- **Tailwind CSS**
- **VS Code**

---

## 📦 Setup
```bash
git clone https://github.com/Ashadeep2002/Twitter_clone_TailwindCSS.git
cd Twitter_clone_TailwindCSS
npm install
npm run build
